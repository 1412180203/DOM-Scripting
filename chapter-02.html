2.2 数组

在 JavaScript 中，数组可以用关键字 Array 声明。例如：

var beatles = Array(); //声明一个数组

var beatles = Array(4); //声明一个大小为4的数组

var beatles = Array("John", "Paul", "George", "Ringo"); //声明数组并对它进行填充

var beatles = ["John", "Paul", "George", "Ringo"]; //声明数组并对它进行填充

数组还可以包含其他的数组，数组中的任何一个元素都可以把一个数组作为它的值：

var lennon = ["John", 1940, false];
var beatles = [];
beatles[0] = lennon;

可以使用数组的 array.length 属性来获得数组里的元素的个数。

2.3 对象

对象也是使用一个名字表示一组值，对象的每个值都是对象的一个属性，创建对象使用 Object 关键字。例如：

var lennon = Object();
lennon.name = "John";
lennon.year = 1940;
lennon.living = false;

创建对象还可以使用花括号：

var lennon = {
   name: "John",
   year: 1940,
   living: false
};

对象是自包含的数据集合，包含在对象里的数据可以通过两种形式访问——属性和方法：属性是隶属于某个特定对象的变量；方法是只有某个对象才能调用的变量。在 JavaScript 里，属性和方法都使用“点”语法来访问。
实例是对象的具体个体。为给定对象创建一个新实例需要使用 new 关键字，例如：

//假设已经存在 Person 对象
var jeremy = new Person; //创建 Person 对象的一个实例
