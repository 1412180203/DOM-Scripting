11.1 视频

在浏览器显示 <video> 元素时，会为其添加一些与浏览器样式统一的标准播放控件。要想自定义这些控件的外观，或者添加新的控件，可以通过一些 DOM 属性来实现，主要包括：
(1) currentTime，返回当前播放的位置，以秒表示。
(2) duration，返回当前媒体的总时长，以秒表示，对于流媒体返回无穷大。
(3) paused，表示媒体是否处于暂停状态。
此外，还有一些与特定媒体相关的事件，可以用来触发你的脚本：
(1) paly，在媒体播放开始时发生。
(2) pause，在媒体暂停时发生。
(3) loadeddata，在媒体可以从当前播放位置开始播放时方式。
(4) ended，在媒体已经播放完成而停止时发生。
使用这些及其属性和事件，可以轻松地创建自定义的视频控件，实现对视频的各种控制。不管创建什么控件，都别忘了在 <video> 元素中添加 control 属性：<video src="movie.ogv" control>

11.2 表单

检查浏览器是否支持某种类型的输入控件：

function inputSupportsType(type) {
   if (!document.createElement) return false;
   var input = document.createElement('input');
   input.setAttribute('type', type);
   if (input.type == 'text' && type != 'text') {
      return false;
   } else {
      return true;
   }
}

检查特定的属性：

function elementSupportsAttribute(elementName, attribute) {
   if (!document.createElement) return false;
   var temp = document.createElement(elementName);
   return (attribute in temp);
}

11.3 HTML5 的其他特性

(1) 使用 localStorage 和 sessionStorage 在客户端存储大型和复杂数据集的更有效方案。
(2) 使用 WebSocket 与服务器端脚本进行开放的双向通信。
(3) 使用 Web Worker 在后台执行 JavaScript。
(4) 标准化的拖放实现。
(5) 在浏览器中实现地理位置服务。

小结

Modernizr (http://www.modernizr.com) 是一个开源的 JavaScript 库，利用它的富特性检测功能，可以对 HTML5 文档进行更好的控制。Modernizr 不会给你添加浏览器不支持的特性，只会为你提供一些不同的 CSS 类名以及特性检测属性。
在文档中嵌入 Modernizr 之后，它会修改 <html> 的 calss 属性，基于可用的 HTML5 特性添加额外的类型，根据这些类名，可以在 CSS 中定义相应的增强和退化版本。
Modernizr 库也提供了 JavaScript 特性检测对象，可以在 DOM 脚本中直接使用，例如检测浏览器是否支持 <canvas>：Modernizr.canvas
