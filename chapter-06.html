6.1 共享 onload 事件

网页加载完毕时会触发一个 onload 事件，这个事件与 window 对象相关联。
如果有多个函数想在页面加载时得到执行，可以使用如下方法：

function addLoadEvent(func) {
   var oldonload = window.onload;
   if (typeof oldonload != 'function') {
      window.onload = func;
   } else {
      window.onload = function() {
         oldonload();
         func();
      }
   }
}
